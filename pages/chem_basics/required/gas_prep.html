氢气(H₂)的制备

反应原理：Zn + H₂SO₄(稀) = ZnSO₄ + H₂↑

实验装置：

  • 发生装置：固液不加热型（启普发生器/简易固液装置）
  • 收集装置：排水法（首选）、向下排空气法

操作要领：

  • 药品选择：锌粒（不用纯锌，反应太慢）+ 稀硫酸（不用盐酸，易挥发）
  • 验纯操作：收集一小试管气体，拇指堵住管口，移近酒精灯火焰，爆鸣声小则纯净
  • 注意通风：氢气易燃易爆，实验时远离火源

注意事项：点燃氢气前必须验纯；不能使用浓硫酸或硝酸制备。

氧气制备实验装置图（此处替换为实际图片）

氧气(O₂)的制备

主要反应原理：

  • 加热固体制备：2KMnO₄ △ K₂MnO₄ + MnO₂ + O₂↑
  • 固液不加热：2H₂O₂ MnO₂ 2H₂O + O₂↑

实验装置：

  • 加热型：固固加热型发生装置 + 排水法/向上排空气法收集
  • 不加热型：固液不加热型发生装置 + 排水法/向上排空气法收集

操作要领：

  • 加热高锰酸钾：试管口略向下倾斜，管口放棉花，先预热再集中加热
  • 收集时机：排水法待气泡连续均匀冒出时再收集
  • 验满方法：带火星木条伸到集气瓶口，木条复燃则满

注意事项：加热法制氧气结束后先移导管后熄灯，防止倒吸。

二氧化碳制备实验装置图（此处替换为实际图片）

二氧化碳(CO₂)的制备

反应原理：CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑

实验装置：

  • 发生装置：固液不加热型（启普发生器/简易装置）
  • 收集装置：向上排空气法（能溶于水，不能用排水法）

操作要领：

  • 药品选择：大理石/石灰石 + 稀盐酸（不用稀硫酸，生成CaSO₄微溶覆盖表面）
  • 验满方法：燃着的木条伸到集气瓶口，木条熄灭则满
  • 净化处理：若含HCl杂质，用饱和NaHCO₃溶液洗气

注意事项：不能用碳酸钠粉末，反应速率太快不易控制。

氯气制备实验装置图（此处替换为实际图片）

氯气(Cl₂)的制备

反应原理：MnO₂ + 4HCl(浓) △ MnCl₂ + Cl₂↑ + 2H₂O

实验装置：

  • 发生装置：固液加热型（圆底烧瓶+分液漏斗）
  • 净化装置：饱和食盐水（除HCl）→ 浓硫酸（除水）
  • 收集装置：向上排空气法/排饱和食盐水法
  • 尾气处理：NaOH溶液吸收

操作要领：

  • 实验条件：必须用浓盐酸，加热
  • 验满方法：湿润的淀粉KI试纸放瓶口，试纸变蓝则满
  • 安全防护：在通风橱中进行，氯气有毒

注意事项：尾气必须处理，防止污染空气；多余氯气用NaOH溶液吸收。

氨气制备实验装置图（此处替换为实际图片）

氨气(NH₃)的制备

反应原理：2NH₄Cl + Ca(OH)₂ △ CaCl₂ + 2NH₃↑ + 2H₂O

实验装置：

  • 发生装置：固固加热型（大试管）
  • 干燥装置：碱石灰（不能用浓硫酸、无水CaCl₂）
  • 收集装置：向下排空气法（管口塞棉花，防止对流）
  • 尾气处理：稀硫酸吸收

操作要领：

  • 试管要求：试管口略向下倾斜，防止冷凝水倒吸
  • 验满方法：湿润的红色石蕊试纸放瓶口，试纸变蓝则满
  • 干燥剂选择：只能用碱性干燥剂

注意事项：不能用NH₄NO₃或NH₄HCO₃，易分解产生杂质气体。

二氧化硫制备实验装置图（此处替换为实际图片）

二氧化硫(SO₂)的制备

反应原理：Na₂SO₃ + H₂SO₄(较浓) = Na₂SO₄ + SO₂↑ + H₂O

实验装置：

  • 发生装置：固液不加热型（圆底烧瓶+分液漏斗）
  • 净化装置：浓硫酸（除水）
  • 收集装置：向上排空气法（易溶于水，不能用排水法）
  • 尾气处理：NaOH溶液吸收

操作要领：

  • 药品选择：亚硫酸钠固体 + 较浓硫酸
  • 验满方法：湿润的品红试纸放瓶口，试纸褪色则满
  • 安全防护：在通风橱中进行，SO₂有毒且有刺激性气味

注意事项：尾气必须处理，防止污染空气；SO₂具有漂白性和还原性。

乙炔制备实验装置图（此处替换为实际图片）

乙炔(C₂H₂)的制备

反应原理：CaC₂ + 2H₂O = Ca(OH)₂ + C₂H₂↑

实验装置：

  • 发生装置：固液不加热型（用分液漏斗控制加水速率）
  • 净化装置：CuSO₄溶液（除H₂S、PH₃等杂质）
  • 收集装置：排水法（微溶于水，不用排空气法）

操作要领：

  • 反应控制：用饱和食盐水代替水，减慢反应速率
  • 验纯操作：乙炔易燃易爆，点燃前必须验纯
  • 杂质去除：电石中含杂质，生成的乙炔需用CuSO₄溶液净化

注意事项：不能用启普发生器（反应剧烈且CaC₂易结块）；远离火源。
